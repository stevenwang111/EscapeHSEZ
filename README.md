# EscapeHSEZ
一个python小游戏

灵感来源于班上一同学制作的游戏，原名“逃离魔爪”，受到班内同学的一致好评。但是原版是用scratch做的，扩展性不高，我打算用Python复刻一个。

## 功能

- [X] 随机读取背景图片
- [ ] 点击按钮切换角色
- [ ] 画格子
- [ ] 通过WASD移动角色
- [ ] 通过屏幕上的按钮移动角色
- [ ] boss移动
- [ ] 失败判定
- [ ] 分数计算

## 其他会加入的功能
- [ ] 最高记录保存
- [ ] 配置文件读取设置
